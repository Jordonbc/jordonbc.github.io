Hardware

I have built several computers in the past. On this webpage I will show you how to build computers and my specifications of all my devices.

Required Parts for a computer to Function:

  • Central Processing Unit
  • Random Access Memory
  • Graphics Card (Can be integrated)
  • Storage (Hard Disk Drive or Solid State Drive)
  • Motherboard
  • Case
  • Thermal Paste
  • Heat-Sink
  • Central Processing Unit Fan
  • Case Fan
  • Mouse
  • Keyboard
  • Monitor or Television
